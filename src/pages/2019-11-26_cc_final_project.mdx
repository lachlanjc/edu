The primary issue I worked on is that last commit (plus the later [fix ranking](https://github.com/lachlanjc/gunfunded/commit/71063548d8aca8da8bafad2aa9950201ceff7c91)). Demoing the project for the first day of the show, I noticed a number of profiles showed no funding data. Eventually, I realized they all had multi-word last names. Because I‚Äôm using two different data sources (theunitedstates.io and OpenSecrets) and matching names to collate the funding data, I discovered they categorized the name parts differently (as last vs first). I solved by getting the last word of the last name. Then came names with capitalized letters in the middle‚Äîso had to lower-case both, in addition to removing accents, standardizing quotes, etc. Finally, two exception to the rule was names starting with "De" & "Des," of which there are 2‚Äîone source had a space after these, the other did not. I wrote a custom regex for this exception, which is insane. Look so simple on the outside!

I‚Äôm rather proud of the [About page](https://gunfunded.com/about)‚Äîfairly succinctly, it explains a lot of the data on the site and how to interpret it. I modeled it directly after the questions I kept answering and what I said during demos while presenting the project at the show, so I hope it‚Äôs helpful to people.

## Behind the scenes: shareable image profiles

_Cross-posted to [notebook.lachlanjc.me](https://notebook.lachlanjc.me/2019-12-11_shareable_profiles_for_gun_funded/)_

![A grid of 4 examples of the social cards](https://d2wkqk610zk1ag.cloudfront.net/items/1u1c1M1j0L2B04082U3O/30jf53wht60447457zcr123c7maehrp4.png)

Sharing links to Congresspeople‚Äôs profiles on Twitter, Facebook, etc, I wanted to highlight the gun funding amount before people even open the site. But I also wanted the profiles to be shareable on Instagram Stories (for us youths), which requires a vertical format (as seen on the left & right). Here‚Äôs how I made the social cards service.

### Avatars

Before we can get to the cards, each Congress member needs an avatar.

The [unitedstates.io](https://theunitedstates.io/) project has the [unitedstates/images](https://github.com/unitedstates/images) archive on GitHub, but it‚Äôs unfortunately out of date. [Someone forked it](https://github.com/SeanKMcGinnis/images) with updates, then I [added](https://github.com/SeanKMcGinnis/images/pull/1) a few missing members.

Next, I wanted to compress the images (they‚Äôre also used on the main website, where Google PageSpeed flagged their extra size as slowing down the site). I wrote a quick Bash script to run Google‚Äôs fantastic [Guetzli](http://github.com/google/guetzli) image compressing library on the folder. (There‚Äôs probably a better way to do this, but I‚Äôm terrible at Bash.)

```sh
#!/bin/bash

for file in $@
do
  guetzli $file $file
done
```

It took about half an hour & drained half my laptop‚Äôs battery, fans up 100% üòÖ

#### Getting online

I‚Äôve been using & loving [ZEIT Now](https://now.sh) for the main Gun Funded site, so I deployed the static folder to a new Now project:

One more command (`now alias gunfunded-avatars.now.sh avatars.gunfunded.com`) and our images are live!

### Social cards

Back to the original mission. I‚Äôve written in the past about [making custom social card services](https://notebook.lachlanjc.me/2019-11-05_custom_social_card_services/) based on [ZEIT‚Äôs og-image](https://github.com/zeit/og-image), but this project is more complicated.

- I need to fetch data from the Gun Funded records database, to show funding
- I need to change styling based on the data
- I want variants for Twitter/Facebook/Open Graph social cards as well as vertical Instagram Stories

I cloned my [ima-cards](https://github.com/lachlanjc/ima-cards) project, and removed [the web testing UI](https://ima-cards.lachlanjc.now.sh) code to simplify development.

The development process took several hours but wasn‚Äôt too hard--mostly, I don‚Äôt have much experience using TypeScript, but running the project with [`now dev`](https://zeit.co/blog/now-dev) was great. I copied the full data JSON file directly over, then worked on the design.

![Editing the template TypeScript file in VSCode on a MacBook Pro](https://d2wkqk610zk1ag.cloudfront.net/items/040A3L3H2H0G3v1N200h/ehp65gdmc7knyf0e90v5ch53wrr4nk20.png)

The design references the "I Voted" stickers, but with the brand, candidate, and funding amount. For Instagram Stories, I used the Instagram handle (@gunfunded), whereas on the web it uses the domain (gunfunded.com). With the Stories‚Äô extra vertical space, it also shows the current term of the Congressperson.

I copied the same color palette as the site to keep the vibe/brand consistent. Though the site uses San Francisco/system-ui for typography, the incredible [Inter](https://rsms.me/inter/) font stepped in here.

Deployment was just as easy as for Avatars‚Äîjust `now` & `now alias` to `cards.gunfunded.com`.

#### End results

Whether for Representatives or Senators, funded or not funded, Republican or Democrat, Instagram or Twitter, cards for all!
